# Usage

Usage: guess [OPTION...] 
Program for guessing number from 1 to 100.

- -c, --convert              Convert roman number to arabic
- -r, --roman                Print romain numbers
- -?, --help                 Give this help list
-     --usage                Give a short usage message
- -V, --version              Print program version
